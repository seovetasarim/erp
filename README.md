# DijitalERP Web Sitesi

DijitalERP stok yönetim yazılımını tanıtan kurumsal web sitesi.

## Teknolojiler

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React (ikonlar)

## Çalıştırma

```bash
# Geliştirme
npm run dev

# Production build
npm run build

# Production başlat
npm run start
```

## Yayınlama

- Vercel, Netlify veya herhangi bir Node.js hosting'e deploy edebilirsin
- `dijitalerp.com.tr` domain'ini bu projeye yönlendir
